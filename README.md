## Installation
Use the `install.sh` script after downloading this repo to install the scripts to `/usr/local/bin` to run the scripts globally in the terminal.

1. Make `install.sh` executable
```bash
chmod +x install.sh
```
2. Run the script
```bash
./install.sh
```
